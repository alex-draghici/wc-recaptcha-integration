=== WooCommerce reCAPTCHA Integration ===
Requires at least: 3.0.1
Tested up to: 3.4
Stable tag: 4.3

Adds Google reCAPTCHA to WooCommerce login and register pages without affecting the site performance.

== Description ==

Adds Google reCAPTCHA to WooCommerce login and register pages.

== Installation ==

1. Upload `wc-recaptcha-integration.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure in Woocommerce -> General
